# Web Fetcher

Smart web content fetcher for Claude Code. Fetches articles and downloads videos from WeChat, Feishu, Bilibili, and more.

## Install

```bash
git clone https://github.com/inspirai-store/web-fetcher ~/.claude/skills/web-fetcher
```

## Quick Start

```bash
python3 ~/.claude/skills/web-fetcher/fetcher.py "URL" -o ~/docs/
```
